Файл для конфигурации представляет из себя список папок для анализа (каждая с новой строки).
Допускаются комментарии в виде строки начинающейся с символа #, такая строка пропускается (можно временно отключать папки)

Запуск программы:
hddclean <полный путь к файлу конфиграции> <удалить файлы старше дней (с момента создания файла)> <удалять мимо корзины - любое булево значение)
